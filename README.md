Code and data for the paper:

"Stochastic Thermodynamics for Autoregressive Generative Models: A Non-Markovian Perspective" (arXiv:2604.07867)
by Takahiro Sagawa (The University of Tokyo)

http://arxiv.org/abs/2604.07867


## Overview

This repository contains:

- GPT-2 experiments: Code for evaluating token-level and block-level stochastic entropy production on GPT-2, including Monte Carlo sampling (Section 5.1) and evaluation on externally prepared causal/non-causal texts (Section 5.2).
- Linear Gaussian (Kalman) experiments: Code for the analytical entropy production and its numerical verification via Monte Carlo sampling (Section 6).
