Generate two sets of short English texts, 100 texts in each set.
Each text must consist of exactly 4 short, simple sentences
(roughly 10 to 15 words each), and the last sentence must
end with a period.

Rules for BOTH sets: Every sentence must be fully understandable
on its own. Do not use pronouns or demonstratives (it, they, she,
he, her, them, this, those, etc.). Write every sentence in the
simple past tense. Each text is about one main person or object.
Mention the main entity in every sentence, always with the same
full noun phrase (e.g., "the old bridge"); never write "a bridge"
first and "the bridge" later. Do not use words that link sentences,
such as "then", "next", "later", "finally", "afterwards", "also",
"so", "as a result". Use 100 different main entities, and use the
same main entity, with the same noun phrase, for causal_texts[i]
and noncausal_texts[i].

Set A ("causal"): Each text is a short narrative describing four
events involving the main entity in temporal/causal order, one
after another. Do not mark the order with dates, weekday names,
or clock times.

Example (use this as the first entry of causal_texts):
"A sudden storm damaged the old bridge across the river. Engineers closed the old bridge to all traffic. Workers repaired the old bridge with new steel beams. Cars and trucks crossed the old bridge without any trouble."

Set B ("non-causal"): Each text consists of 4 independent factual
statements about lasting features of the main entity. The statements
should have no temporal, causal, or discourse dependency on each
other, and their order should be freely exchangeable. Avoid listing
items in a conventional ascending or descending order (e.g., by
size, date, or number).

Example (use this as the first entry of noncausal_texts):
"The old bridge spanned a wide river with three stone arches. The old bridge carried a narrow road between two villages. The old bridge appeared on many postcards of the valley. The old bridge attracted photographers from distant towns."

Output the result as a JSON object with the following structure:
