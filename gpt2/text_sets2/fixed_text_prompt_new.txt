Generate two sets of short English texts, 100 texts in each set. Each text must consist of exactly 4 short, simple sentences (roughly 5 to 15 words each), and the last sentence must end with a period.

#IMPORTANT: For both sets, all four sentences in each text must share the same grammatical subject (i.e., every sentence must begin with the same noun phrase). 

#Set A ("causal"): Each text should be a short narrative describing a sequence of events in temporal/causal order. The 4 sentences should describe things that happen one after another. Do not mark the order with dates, weekday names, or clock times.

Example (use this as the first entry of causal_texts):
"The glass slipped from Alice's hand. The glass fell to the floor. The glass broke into many pieces. The glass left a wet stain on the tile floor."


#Set B ("non-causal"): Each text should consist of 4 independent one-time events. The events should have no temporal, causal, or discourse dependency on each other, and their order should be freely exchangeable. In particular, avoid listing items in a conventional ascending or descending order (e.g., by size, date, or number).

Example (use this as the first entry of noncausal_texts):
"The library lent a heavy atlas. The library hosted a chess tournament. The library displayed old framed photographs. The library received a donated wooden shelf from a volunteer."


#Additional rules for both sets:
- Write every sentence in the simple past tense. Do not use pronouns or demonstratives (it, they, she, he, her, them, this, those, etc.). If the same person or object appears in more than one sentence, use exactly the same full noun phrase each time (e.g., "Alice", "the glass"); never write "a glass" first and "the glass" later. 
- Do not use words that link sentences, such as "then", "next", "later", "finally", "afterwards", "also", "so", "as a result".
- Use a period only at the end of each sentence; do not use abbreviations containing periods (such as "Mr.", "Dr.", or "U.S."), initials (such as "J. K."), or decimal numbers (such as "3.5").


Output the result as a JSON object with the following structure:
{
  "causal_texts": [<list of 100 strings>],
  "noncausal_texts": [<list of 100 strings>]
}

Output only the JSON with no other text.
