Generate two sets of short English texts, 100 texts in each set.
Each text must consist of exactly 4 short, simple sentences
(roughly 10 to 15 words each), and the last sentence must
end with a period.

Additional rules for BOTH sets: Every sentence must be fully
understandable on its own. Write every sentence in the simple past
tense. Do not use pronouns or demonstratives (it, they, she, he,
her, them, this, those, etc.). If the same person or object appears
in more than one sentence, use exactly the same full noun phrase
each time (e.g., "Maria", "the glass"); never write "a glass" first
and "the glass" later. Do not use words that link sentences, such
as "then", "next", "later", "finally", "afterwards", "also", "so",
"as a result".

Set A ("causal"): Each text should be a short narrative describing
a sequence of events in temporal/causal order. The 4 sentences
should describe things that happen one after another. Do not mark
the order with dates, weekday names, or clock times.

Example (use this as the first entry of causal_texts):
"The glass slipped from Maria's hand. The glass fell to the floor. The glass broke into many pieces. Maria swept the floor carefully."

Set B ("non-causal"): Each text should consist of 4 independent
one-time events on a common theme. The events should have no
temporal, causal, or discourse dependency on each other, and their
order should be freely exchangeable. In particular, avoid anaphoric
references between sentences (e.g., "also", "the above", "all
three") and avoid listing items in a conventional ascending or
descending order (e.g., by size, date, or number).

Example (use this as the first entry of noncausal_texts):
"A student borrowed a heavy atlas from the town library. The town library hosted a small chess tournament in the main hall. A tourist photographed the marble staircase of the town library. A volunteer repaired a wooden shelf at the town library."

Output the result as a JSON object with the following structure:
{
  "metadata": {
    "model": "<your model name and version>",
    "date": "<today's date and year>",
    "note": "Output used as-is without manual revision or selection."
  },
  "causal_texts": [<list of 100 strings>],
  "noncausal_texts": [<list of 100 strings>]
}
Output only the JSON with no other text.
